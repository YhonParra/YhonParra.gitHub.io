Actividad GitHub


• Configurará una cuenta de usuario en la plataforma www.github.com
• Instalar el cliente de GIT para escritorio en su computador.
• Instalar el cliente de GITHUB para escritorio en su computador.
• Crear un repositorio de prueba el cual debe estar sincronizado con el cliente de escritorio y su cuenta local de GITHUB.
• Creará un archivo con el nombre index.html y lo subirá desde su computador al repositorio local de GITHUB para luego sincronizarlo con el remoto ( www.github.com).
• Publicará mediante documento PDF el link de su repositorio creado en GITHUB. Para su respectiva revisión.
Link de descarga Git: https://git-scm.com/download/win
